Веб-приложение для учёта матчей и отображения рейтинга ELO для StarCraft TMG.

## О проекте

TMG ELO – это сайт для:
- отправки результатов матчей
- просмотра общего рейтинга игроков
- просмотра списка сыгранных матчей
- просмотра профиля игрока
- административного редактирования игроков и матчей

Проект рассчитан на локальный запуск и дальнейшее развёртывание на сервере.

## Основные возможности

- рейтинг ELO для матчей 1v1
- автоматическое создание игрока при первом матче
- выбор расы для обоих игроков
- определение приоритетной расы игрока по количеству сыгранных матчей
- таблица рейтинга
- список game reports
- профиль игрока со статистикой
- административный режим
- редактирование игроков
- редактирование и удаление матчей
- пересчёт рейтинга после изменения матча
