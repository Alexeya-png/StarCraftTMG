Ranked game guidelines

There is no separate strict ruleset for ranked games at the moment. The main idea is simple – respect your opponent and play as fairly as you would in a regular friendly game.

Ranked Games

How ranked match reporting works

Roster format

Open or closed roster is up to both players. Open roster is recommended because it is easier to review your opponent’s army before the game starts and catch possible roster mistakes in advance.

Submitting results

After the match, please use the TTS module to submit the result whenever possible. If that is not available, the winner must submit the match manually.

Player names and aliases

Use the same player names as on the Koprulu Sector Discord server when submitting a match.

If your Steam name is different, send it through the Contact Admin form. The site supports alternate names, so Steam nicknames can still be linked to your main Discord profile.

Questions and profile setup

For any ranked-related questions, contact Alexey'a.

You can ask to personalize your profile, for example by adding a country flag or a Discord link.

Ranked TTS module

- Put the mission card on the submit board.
- Set the final score and enable Ranked only for ranked games.
