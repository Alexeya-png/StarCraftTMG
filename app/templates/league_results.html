League rules

How the league works

Main idea

A new league starts when a new wave of miniatures is released. The site will show the current league and the previous league, so players can compare the active season with the last completed season.

League standings use only rated matches from the selected league. Unrated matches do not affect league points, race leaders, activity leaders, badges or league colours.

Win3 points
Draw1 point
Loss-1 point

These are the normal values. Head-to-head rules can change the points when one player already leads the same opponent by 2 or more wins.

Opponent activity requirement

Points are counted only when the opponent has played more than 3 rated matches in the same league.

If the opponent has 1–3 rated matches, the match still counts for games, record and winrate, but gives 0 league points. Once the opponent reaches 4 rated matches, points can be counted normally.

Head-to-head favorite rule

If one player leads the same opponent by 2 or more wins, that player becomes the head-to-head favorite in that matchup.

The favorite gets +1 point for a win and -2 points for a loss. The trailing player keeps the normal +3 points for a win and -1 point for a loss.

Draws against a favorite

If a player leads the same opponent by 2 or more wins, a draw gives 0 points to the favorite and +1 point to the trailing player.

Example: Player A leads Player B 2:0. If their next game is a draw, A gets 0 points and B gets +1 point.

Badges

The top player of each race in a league can receive a race season badge. During the current season, race leaders receive a Contender badge for their faction. After the season ends, the final race leaders receive Champion badges for their faction. Badges are visible on the player profile.

Mini FAQ

Do league points change ELO?

No. ELO and league points are separate. ELO is the rating system. Points are used only for league standings.

Do unrated games count?

No. League results use rated matches only.

Why can a win give 0 points?

A win can give 0 points when the opponent has not played enough rated matches in the league. Head-to-head favorites now get +1 point for a win instead of 0.

Why can a win give only 1 point?

If a player already leads the same opponent by 2 or more wins, that player is the head-to-head favorite and gets only +1 point for another win.

When are badges final?

Current league badges and colours reflect current results. Final season badges are awarded when the league is completed.
